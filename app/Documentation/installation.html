Installation Steps

Step-1: Copying Tradebox Installation Files To Server

php greater than 7.4 and 8 also can use | MySQL (5.1+) | Apache 4

php extension must be enabled : intl | mbstring | json

  1. Extract Tradebox-v6.4.zip file.
    You will then get two files:
    • Documentation
    • Tradebox-v6.4.zip
  2. Upload the Tradebox-v6.4.zip file to your server using cPanel or FTP.
    The example below shows how to upload and ready our Tradebox-v6.4 installer on CPanel:
    [Note: These steps will be same for all other systems.]
    • Click on File Manager.

    • Open public_html folder by double-clicking on the public_html icon.

    • Click on +Folder in the top menu to create a new folder in your currect directory.

    • Enter tradebox as the folder name in the feild and then click on Create New Folder.

    • Now click on the Upload button in the top menu.

    • We will now upload the installer file:
      1. Click on Select File.
      2. Select the tradebox-v6.4.zip installer zip file.
      3. Click on Open button.
      4. Click on the Go Back link found in the bottom of the page.



    • We will now extract the uploaded zip file:
      1. Select the tradebox-v6.4.zip file.
      2. Click on Extract.



    • Click on Extract Files button.

    • Click on Close button.

    • Now we will delete the tradebox-v6.4.zip file.



    • Lastly we will move our files to the /public_html" tradebox-v6.4.zip file.




Step-2: Creating a Database & Database Administrator

  1. In your cPanel > Database (section) > Click on MySQL Databases



  2. We will first create a new database. Enter a new database name and then click on Create Database button.



  3. Now create an administrator for that database. Enter the username and password for the database administrator and then click on Create User



  4. We will now add the Administrator to the Database and give it all priviliges.





Step-3: Installing Tradebox (Using Bdtask Software Installer)

  1. Open the installer in your browser by searching http://yourdomain.com/install Then click on Next.
    [Note: Replace "yourdomain" with your domain name.]

    Then enter your Envato User ID and Purchase Key




  2. Now make sure the mentioned files and directories have their permission set to writable.



    Then make sure your server meets all the requirements needed to run Tradebox. (Note: If the requriements are not met then please make sure the requirements are met before proceeding with the installation). Click on the Next button.




  3. Now enter your Database Name, Database Username, Database Username's Password and your Hostname.

    Then click on the Next button.




  4. Enter your Admin Username and Password. (Note: These will be used to login to the admin panel of Tradebox)




  5. Click on the Finish Installation Process button.




  6. You must now delete the installer by click on the Delete Now button for the application to run.




  7. Click on Launch Now button to launch Tradebox.




Thank you for purchasing the Tradebox, if you have any questions, please contact us at info@bdtask.com

You're Done!
