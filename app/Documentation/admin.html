❑ Finance

  1. Finance (Fiat Withdraw List)

    All fiat withdraw transaction requests will be show here. This includes success, failed as well as pending withdrawal requests.

    For any pending withdrawal request, the admin will find and complete the withdrawal requests by using the Confirm or Cancel buttons under the Action section.



  2. Finance (Crypto Withdraw List)

    All crypto withdraw transaction requests will be show here. This includes success, failed as well as pending withdrawal requests.

    For any pending withdrawal request, the admin will find and complete the withdrawal requests by using the Confirm or Cancel buttons under the Action section.



  3. Finance (Deposit List)

    All fiat and crypto deposit transactions will be shown here. This includes success, failed as well as pending deposit requests.

    For any pending deposit request, the admin will find and complete the deposit requests by using the Confirm or Cancel buttons under the Action section.



  4. Finance (Add Credit)

    Here the admin can send funds (or credit) to any user account.



  5. Finance (Credit List)

    History of credits transfered to any user account by the admin can be seen here.

❑ Exchange

  1. Exchange (Currency)

    This page list all fiat and crypto currencies that are in the exchange. The admin can add new currencies, edit existing currencies or remove any currency in the exchange from this page.



    To add a new currency the admin must:
    • Select the Currency Type
    • Enter a Coin ID (Note: The ID will automatically be set auto-incrementally.)
    • Enter the coin Symbol.
    • Enter the Coin Name.
    • Enter the Coin Full Name.
    • Enter the Rank. (Compared to other currencies)
    • Choose whether or not to show the currency in the homepage.
    • Give it a serial number.
    • Upload the coin image.
    • Set the coin to be active or inactive.
    • Click on Create button.



    When the admin chooses to update a currency they will be brought to this page. After making the needed change they must click on Update button to update the currency.



  2. Exchange (Market)

    Here the admin can add a new Market or manage previously created Markets.



    If the admin chooses to create a new Market they will be brought to this page. After filling in the feilds they must click on Create button to create the new market.



    If the admin chooses to edit an existing Market they will be brought to this page. After making the needed change they must click on Update button to update the existing market.



  3. Exchange (Coin Pair)

    Here the admin can create a new coin pair or manage previously created coin pairs.



    If the admin chooses to create a new coin pair they will be brought to this page. After filling in the feilds they must click on Create button to create the new coin pair.



    If the admin chooses to edit an existing coin pair they will be brought to this page. After making the needed change they must click on Update button to update the existing market.
